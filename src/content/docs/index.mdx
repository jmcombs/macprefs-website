## Why macprefs?

<CardGrid stagger>
  <Card title="JSON Config" icon="document">
    Single file defines all your preferences. Human-readable,
    version-controllable, shareable.
  </Card>
  <Card title="Drift Detection" icon="magnifier">
    See exactly what will change before applying. Plan before you commit.
  </Card>
  <Card title="Instant Rollback" icon="backward">
    Undo any change with one command. Every apply is automatically backed up.
  </Card>
  <Card title="Fast & Native" icon="rocket">
    Pure Swift, universal binary. Process 100+ domains in under 10 seconds.
  </Card>
  <Card title="No Elevation" icon="approve-check">
    User-level preferences only. No sudo required, no system modifications.
  </Card>
  <Card title="CI/CD Ready" icon="setting">
    Headless mode for automation. Perfect for fleet management and testing.
    (Pro)
  </Card>
</CardGrid>
