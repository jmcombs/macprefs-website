Common Finder preference configurations.

## Show File Extensions

```json
{
  "version": "1.0",
  "domains": {
    "com.apple.finder": {
      "AppleShowAllExtensions": true
    }
  }
}
```

## Show Hidden Files

```json
{
  "version": "1.0",
  "domains": {
    "com.apple.finder": {
      "AppleShowAllFiles": true
    }
  }
}
```

## Default View Mode

```json
{
  "version": "1.0",
  "domains": {
    "com.apple.finder": {
      "FXPreferredViewStyle": "clmv"
    }
  }
}
```

Options:
- `"icnv"` — Icon View
- `"Nlsv"` — List View
- `"clmv"` — Column View
- `"glyv"` — Gallery View

## Show Path Bar

```json
{
  "version": "1.0",
  "domains": {
    "com.apple.finder": {
      "ShowPathbar": true
    }
  }
}
```

## Disable Extension Change Warning

```json
{
  "version": "1.0",
  "domains": {
    "com.apple.finder": {
      "FXEnableExtensionChangeWarning": false
    }
  }
}
```

## Sidebar Favorites (Unordered Compare)

```json
{
  "version": "1.0",
  "domains": {
    "com.apple.finder": {
      "FavoriteItems": {
        "value": ["Applications", "Documents", "Downloads"],
        "compare": "unordered",
        "note": "Sidebar items - order doesn't matter"
      }
    }
  }
}
```

## Full Finder Configuration

```json
{
  "version": "1.0",
  "domains": {
    "com.apple.finder": {
      "AppleShowAllExtensions": true,
      "ShowPathbar": true,
      "FXEnableExtensionChangeWarning": false,
      "FXPreferredViewStyle": "Nlsv"
    }
  }
}
```
